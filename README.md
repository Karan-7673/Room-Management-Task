Start Metro
Metro is the JavaScript bundler used by React Native.

Using npm
npm start

OR using Yarn
yarn start

Run the App
Open another terminal in the project root and use the commands below.

Android npm run android

or
yarn android

iOS (macOS only)

First time only (or when dependencies change):

bundle install bundle exec pod install

Then run:

npm run ios

or
yarn ios

If everything is configured correctly, the app will launch on your simulator or device.
